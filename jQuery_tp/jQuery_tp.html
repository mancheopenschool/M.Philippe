Créer des éléments HTML

Création d'une variable

$h1 = $("<h1>Titre</h1>");

$p = $("<p>Paragraphe</p>");

Insérer des éléments HTML .append - .prepend

.append: insère l'élément spécifié en dernier enfant de l'élément cible.
.prepend()- insère l'élément spécifié en premier enfant de l'élément cible.


$("body").append("<p>Je suis un paragraphe</p>")

Avant - Après

Pour insérer un paragraphe après l'id #un

$('#un').after("<p>Un louveau paragraphe</p>");

Supprimer des éléments .empty() - .remove()

.empty() - supprime le contenu d'un élément et tous ses descendants.
Par exemple, si vous videz (avec .empty()) un 'ol', vous allez aussi supprimer tous ses 'li' et leur texte.

.remove() - supprime non seulement le contenu d'un élément, mais également l'élément lui-même.

$par = $("<p>Un nouveau paragraphe</p>");
$par.remove();

Faire basculer les classes .toggleClass()

.toggleClass() - Si l'élément appelé a la classe reçue en argument, alors .toggleClass() retire cette classe ; si l'élément cible n'a pas cette classe, .toggleClass() l'ajoute.

$("#text").click(function(){
$(this).toggleClass("highlighted"); });

Modifier le contenu .html() - .val()

.html() - est utilisé pour définir le contenu du premier élément correspondant trouvé.
$("p").html("La magie de jQuery en action !");

.val() - est utilisé pour récupérer la valeur d'éléments d'un formulaire.
$('input:checkbox:checked').val();

$("p").html("La magie de jQuery en action !");

Survol

l'effet hover est capable de prendre deux function(){}
séparées par une virgule essntielle.
Le premier function(){} que nous passons sera lancé quand nous survolons la cible. Là, nous appliquons une classe miseenvaleur.
La seconde function(){} sera appelée quand notre souris quitte l'objet. C'est à ce moment que nous enlevons notre classe miseenvaleur.

$('div' ).hover(function(){
$('div').addClass('active')
},
function(){
$('div').removeClass('active')
}
);
